API Documentation

Integrasikan layanan email sementara Tempbox ke dalam aplikasi Anda.

Informasi Dasar

Semua request menggunakan method GET dan mengembalikan respons dalam format JSON. CORS diizinkan untuk semua domain.

Base URL: https://tempbox-ivory.vercel.app/api

GET /generate

Membuat alamat email acak baru.

{
  "success": true,
  "result": {
    "email": "example@domain.com",
    "emailStatus": "good",
    "uptime": 10
  }
}
GET /inbox?email={email}

Mengambil pesan dari inbox.

{
  "success": true,
  "result": {
    "inbox": [
      {
        "from": "Sender",
        "subject": "Hello",
        "message": "Content here...",
        "created": "20 Feb 2026 19:00"
      }
    ]
  }
}
GET /validate?email={email}

Validasi status aktif domain email.
